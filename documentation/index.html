The Clipboard Project

About

Clipboard by Jackson Huff is a free and open source software power tool that lets you cut, copy, and paste anything, anywhere, all in the terminal!

Previously, using the clipboard from the terminal was an inconsistent, user-unfriendly mess. Clipboard changes all of this by unifying all the different clipboard tools into a single, highly polished one.
