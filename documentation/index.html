The Clipboard Project Cut, copy, and paste anything, anywhere!

Clipboard by Jackson Huff is a power tool that lets you cut, copy, and paste anything, anywhere, all in the terminal.

It's one of those tools you'd never believe you went without after trying out.

What's the big deal?

If you're a terminal user, maybe you've encountered situations where...

  • ...you need to copy a file somewhere, but don't know where exactly.
  • ...you need to make a change to something and you want to revert those changes quickly.
  • ...you need to share a file with a website but you only know where this file is in the terminal.
  • ...you need a "scratchpad" to save some words on to help remind you in the future.
  • ...you need to save a bunch of data from some program for later.

Before Clipboard, you'd have to research which specific tool to download and figure out how to use it.
