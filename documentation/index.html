What's Clipboard?

If you use the terminal, maybe...

> you need to copy a file somewhere, but don't know where exactly.
> you need to make a change to something and you want to revert those changes quickly.
> you need to share a file with a website but you only know where this file is in the terminal.
> you need a "scratchpad" to save some words on to help remind you in the future.
> you need to save a bunch of data from some program for later.

Before Clipboard, you'd have to spend time researching which specific tool to download for your platform and figuring out and memorizing how to use it and hoping you get its specific name and syntax right every single time or else you get mysterious errors or nothing at all. Yuck, ugh, puke!

Clipboard changes all of this.

With Clipboard, do anything that involves moving information around way quicker and much simpler. Save time and effort the easy way.
